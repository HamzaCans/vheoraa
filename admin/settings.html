🛡️ Bakım Modu

Siteyi ziyaretçilere kapatıp "Yakında Hizmetinizdeyiz" sayfasına yönlendirir. Yönetici paneli ve API erişime açık kalır.

🔗 Bakım Modu Açıkken Site Erişimi (Sadece Senin İçin)

Bypass token oluşturarak bakım modu açıkken bile siteyi görüntüleyebilirsin. Token sadece senin tarayıcına kaydedilir.

🌐 Siteyi Görüntüle

Durum: Pasif

💰 Altın Fiyatlandırma

Ürün fiyatlarında kullanılan sabit işçilik maliyetini (marj) ayarlayın. Fiyat formülü: (Has Altın + Marj) × Gram.

Has Altın fiyatı + bu değer, ürünün gramı ile çarpılarak satış fiyatı hesaplanır. Örn: 6343 + 2500 = 8843 ₺/gr
